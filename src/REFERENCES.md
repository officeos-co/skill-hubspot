## API Coverage
- **Contacts**: list, get, create, update, delete, search contacts
- **Companies**: list, get, create, update, delete, search companies
- **Deals**: list, get, create, update, delete, search deals
- **Pipelines**: list pipelines, get pipeline, list pipeline stages (deals and tickets)
- **Tickets**: list, get, create, update, delete tickets
- **Associations**: create, list, remove associations between CRM objects
- **Engagement**: create notes, tasks, email activities; list engagements per object
- **Lists**: list contact lists, get list, create list, add/remove contacts from static lists
- **Email**: list marketing emails, get email stats (opens, clicks, bounces)
- **Properties**: list properties per object type, create custom properties
